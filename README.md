`lumier` is an R package that shows the usage of `beautier` by a Shiny app.

![lumier v0.1](pics/lumier_0_1.png)

## Installation

```
devtools::install_github("ropensci/lumier")
```

## Usage

```
library(lumier)
lum_run()
```
